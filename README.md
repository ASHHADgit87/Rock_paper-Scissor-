# Rock Paper Scissors Game

Rock Paper Scissors is a simple interactive web-based game built using **HTML, CSS, and JavaScript**. It allows users to play against the computer with real-time score tracking and a responsive UI.

The project demonstrates fundamental frontend development concepts including DOM manipulation, event handling, and dynamic interface updates.

---

# Features

| Feature | Description |
|---|---|
| Interactive Gameplay | Play Rock, Paper, or Scissors against the computer |
| Score Tracking | Real-time score updates for both user and computer |
| Responsive Design | Works smoothly on mobile, tablet, and desktop devices |
| Instant Feedback | Immediate result display after each move |

---

# Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML |
| Styling | CSS |
| Logic | JavaScript |

---

# Gameplay Logic

| Player Choice | Computer Response | Result Logic |
|---|---|---|
| Rock | Paper / Scissors | Win/Lose based on standard rules |
| Paper | Rock / Scissors | Win/Lose based on standard rules |
| Scissors | Rock / Paper | Win/Lose based on standard rules |

---

# Project Flow

```text
User Selects Move
        ↓
Computer Generates Random Move
        ↓
Winner Calculated
        ↓
Score Updated
        ↓
UI Refreshes Result
```
